# NeckFit Pro v3 — Premium Android App

NeckFit Pro is an offline Android posture and neck-exercise guide app. It focuses on safe posture improvement only. It does **not** claim to increase bone length.

## New premium features added

- Beautiful Indian theme UI with saffron, white, green and navy styling
- Splash screen experience
- 3-screen onboarding
- Exercise image/illustration guide
- Step-by-step how-to instructions
- Benefits, mistakes to avoid and coach tips
- Workout timer with auto-next
- Sound beep and vibration in the last 3 seconds
- Progress screen with streaks and calendar-style completion grid
- Achievement badges
- Hindi/English mode toggle
- Dark mode toggle
- Daily reminder time picker
- Safety warning page inside profile
- GitHub Actions APK build workflow

## How to build APK on GitHub

1. Create a new GitHub repository.
2. Upload all files inside this project folder.
3. Go to **Actions**.
4. Run **Build NeckFit Pro APK**.
5. Download the APK from workflow artifacts.

## How to open in Android Studio

1. Open Android Studio.
2. Choose **Open Project**.
3. Select this project folder.
4. Let Gradle sync.
5. Click **Run** or **Build > Build APK(s)**.

## Safety disclaimer

This app is for posture and fitness guidance only, not medical treatment. Stop exercise if pain, dizziness, numbness, tingling or discomfort occurs. Consult a doctor or physiotherapist if symptoms continue.
